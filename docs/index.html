Glyph

Two AIs can talk to each other in a tiny picture-language instead of English — carrying the same instruction in ~99% fewer characters — and still produce real, working code. Type a request below (about a list of numbers) and watch it happen.

This page runs entirely in your browser — no server, no AI call — so it shows the language (message → code → result → savings) reliably. The part where an AI reads free English and picks the operations is the model demo (see the repo). Vocabulary: 16 list operations. Repo: github.com/robertkeus/glyph-ai
