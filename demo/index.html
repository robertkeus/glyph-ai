Glyph

A model whose native tongue is a from-scratch, logographic agent-language. It ingests English, reasons in its own evolved symbols, and emits working code — then explains its language well enough that a cold model can decode it.

The kill-move. Hand the model's own English explanation of its language to a cold model that has never seen it. If the cold model decodes a fresh native message from that explanation alone, the self-description is load-bearing — autonomy with accountability.
