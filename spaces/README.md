# Glyph — ask in English, it answers in its own symbol language, then working code.
Two agents talk in a 4-byte glyph language (~99% fewer bytes than English); the
output is real, executed Python. Base model + deterministic decode (no training
needed to run). See github.com/robertkeus/glyph-ai.
